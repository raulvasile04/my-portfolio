# Personal Portfolio Website 

This is my personal portfolio website . It has listed my education levels , previous jobs ,
skills and about me section . It was built as a first project to practice HTML ,CSS and JavaScript

Live Demo : https://raulvasile04.github.io/my-portfolio/

## Tech Stack : 
- HTML5
- CSS
- JavaScript
- Tawk.to(Chat Widget)

## Features
- Responsive Layout
- Dark / Light Mode toggle
- Embedded Map
- Navigation System
- Chat Widget

## What I learned
- Basics of programming with HTML , CSS and JavaScript
- Hosting a website using GitHub
- Integrating third-party services

## Run Locally
- Clone the repository
- Open "index.html"
